Google Analytics Bridge

This helper class bridges the event tracking from your app to the web version of google analytics. This will let you use Google Analytics on your OSX application.

// Initialize your GATracker
NSString *trackerID = @"UA-XXXXXX-Y";

self.GATrack = [[GoogleAnalyticsBridge alloc] init:self.window
                                         trackerID:trackerID];

[self.GATrack trackView:@"applicationDidFinishLaunching"];

Installation

Drag Classes/GoogleAnalyticsBridge.* into your repository.

Something you should know

The bridging make use of this repo's project github page. If you're trying to fork your repository, you can update the trackers pageViewURL accordingly.

self.GATrack.pageViewURL = @"yourawesomerepo.github.io/ga-bridge/pageview"
